# Vector rotation

Vector rotation about a given axis and angle after Schaeben et al.
(2024)

## Usage

``` r
vrotate(x, rotaxis, rotangle)
```

## Arguments

- x, rotaxis:

  numeric 3-column matrices representing vectors

- rotangle:

  numeric. Angle in radians

## Value

numeric 3-column matrix
